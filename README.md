# Binary Chain Analyzer

This project analyzes microscopic images of binary colloidal chains composed of:

- 1 µm particles
- 2.8 µm particles

Objectives:

- Detect particles automatically
- Classify particles by size
- Identify chain ends
- Calculate end-particle probabilities
- Study the effect of magnetic field on chain structure
